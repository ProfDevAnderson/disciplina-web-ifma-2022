Conteúdo Lecionado

HTML

Linguagem de marcação utilizada na construção de páginas na Web.

CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web.

Bootstrap

Framework para desenvolvimento de componentes de interface para web

Javascript

Linguagem de programação para adicionar interatividade em páginas web

Spring MVC

Framework para desenvolvimento de aplicações web

Projetos Desenvolvidos

Foram desenvolvidos diversos projetos, utilizando conhecimento adiquirido ao longo da disciplina. Os alunos foram desafiados a criar 4 projetos

  • Criação de um site com HTML e CSS
  • Criação de um site com framework Bootstrap
  • Criação de 3 mini projetos com Javascript
  • Criação de um sistema web com CRUD
